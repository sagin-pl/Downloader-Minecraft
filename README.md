
# Downloader-Minecraft

Plugin do serwerów minecraft służący do pobierania filmów z różnych platfrom.

## Features

- Otrzymywanie linku z filmem na chacie w grze


## Usage

**tiktok**
```
/pobierz [link]
```

**youtube**

```
/pobierz [link] [best/hd]
```
